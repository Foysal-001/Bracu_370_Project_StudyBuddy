## Features

### Login / Logout (User)
This feature allows users to securely log into the system and log out when needed. It ensures that each user's actions are tracked and personalized within the platform.

### Filtering & Search (By Course, Interest)
Users can search and filter study groups based on specific courses and interests. This helps users quickly find relevant groups without manually browsing through all available options.

### Study Group Creation
Users can create new study groups by providing details such as course name and interest/topic. This enables students to form communities for collaborative learning.

### Group Schedule
Each group can have a defined schedule (such as meeting time and date), helping members stay organized and manage their study sessions effectively.

### Updating Group (Joining / Leaving Group)
Users can request to join groups or leave them at any time. This gives flexibility and control over group participation.

### Viewing Group Details and User Details
Users can view detailed information about study groups and other users. This helps them understand group activities and decide which groups to join.

### Approve / Reject Group Joining Request
Group admins have the ability to approve or reject join requests. This ensures better control over group members and maintains group quality.

### Leaderboard
The leaderboard displays user activity or contribution levels, encouraging engagement and healthy competition among users.

### Combine Group with Same Course-Interest
Groups with the same course and interest can be merged together to avoid duplication and create more active and efficient study communities.
